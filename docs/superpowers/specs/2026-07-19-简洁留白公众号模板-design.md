# 简洁留白公众号模板设计

## 目标

在 ObsidianToSM 的公众号排版模板中新增“简洁留白”。它以贴图模式的克制阅读风格为参考，同时继续使用当前选中的颜色主题。

## 范围

- 仅新增一个公众号排版模板，不改动贴图渲染、微信接口或现有模板。
- 模板出现在右侧工作台的“排版模板”下拉框中。
- 模板输出继续使用现有 Markdown 到公众号 HTML 的渲染链路。

## 排版规则

- 正文使用系统中文字体，字号 16px，行高 1.75，段落间距 0.55em。
- 一级、二级、三级标题左对齐，不使用色块或粗边框；二级标题仅保留当前主题色的 1px 下划线。
- 引用块使用极浅灰背景、主题色 3px 左线和紧凑内边距。
- 列表沿用兼容微信的项目段落结构，颜色随正文，不增加额外装饰。
- 表格使用浅灰 1px 细线和白色表头；不使用主题色表头；单元格内边距 0.4em 0.5em。
- 链接、标题下划线与引用左线使用当前颜色主题的强调色。
- 代码块继续复用现有代码高亮、单列行号与空行修复逻辑。

## 兼容性与验证

- 不变更 `renderer.ts` 的列表和代码块兼容逻辑。
- 在 `layouts` 测试中验证模板可发现、强调色生效、表格表头无主题色背景、引用块使用轻量边线。
- 执行全量单元测试与生产构建。
- 构建结果同步到当前 Obsidian Vault 插件目录。
