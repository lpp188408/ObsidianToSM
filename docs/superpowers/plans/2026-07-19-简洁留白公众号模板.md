# 简洁留白公众号模板 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增“简洁留白”公众号排版模板，提供与贴图阅读风格一致的轻量排版。

**Architecture:** 模板仅作为 `LAYOUTS` 中新增的样式工厂，由既有下拉框自动发现。渲染器、微信接口和贴图模式不修改，模板继续接收当前 `WechatTheme` 的强调色。

**Tech Stack:** TypeScript、Marked、Vitest、esbuild、Obsidian 插件 API。

## Global Constraints

- 仅修改公众号布局样式和布局测试。
- 保持现有五个模板的样式与标识不变。
- 正文 16px、行高 1.75；表头白色，表格使用浅灰 1px 细线。
- 必须执行 `npm test` 与 `npm run build`，并同步 `main.js`、`manifest.json`、`styles.css` 到当前 Vault。
